# Minions Projects

Project definitions, variants, goals, status, and ownership

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-projects/sdk`) — Core library for Node.js
- **Python SDK** (`minions-projects`) — Python bindings
- **CLI Tool** (`@minions-projects/cli`) — Command-line interface
- **Web App** — Interactive playground at [projects.minions.wtf](https://projects.minions.wtf)
