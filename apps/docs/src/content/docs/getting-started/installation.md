---
title: Installation
description: Install Minions Projects packages
---

## TypeScript / Node.js

```bash
npm install @minions-projects/sdk minions-sdk
```

## Python

```bash
pip install minions-projects
```

## CLI

```bash
npm install -g @minions-projects/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
